# Blogpost Project

A full-stack blog website where users can create, edit, delete, and view blog posts. The website also includes user authentication, search, and filtering features.

## Highlights

- **CRUD Operations**: Users can create, read, update, and delete blog posts.
- **User Authentication**: Implemented login and logout functionality with secure authentication.
- **Search Functionality**: Allows users to search for specific blog posts based on keywords.
- **Filtering**: Users can filter posts by category or other criteria.
- **Frontend**: Developed using EJS for dynamic templating for UI.
- **Backend**: Built with Node.js, Express, and MongoDB for efficient data management and routing.

## Tech Stack

- **Frontend**: EJS, HTML, CSS, JavaScript
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Authentication**: JWT (JSON Web Tokens)
- **Others**: Method Override, Cookie Parser, CORS, Helmet, and more
